📈 Sistema de Predicción de Ventas

Aplicación basada en el método numérico de Interpolación Lineal

📚 Introducción

Las empresas necesitan anticipar su comportamiento comercial para planificar inventarios, compras y estrategias de crecimiento. Este proyecto utiliza la Interpolación Lineal para estimar ventas futuras a partir de datos históricos.

🧮 ¿Qué es la Interpolación Lineal?

La interpolación lineal es un método numérico utilizado para estimar valores desconocidos a partir de dos valores conocidos. Se basa en la idea de que los datos siguen una tendencia lineal entre dos puntos.

y = y₁ + ((x - x₁)(y₂ - y₁)) / (x₂ - x₁)

⚙️ Procedimiento del Método

  1. Ingresar las ventas de dos meses conocidos.
  2. Seleccionar el mes que se desea estimar.
  3. Aplicar la fórmula de interpolación lineal.
  4. Calcular el valor estimado.
  5. Mostrar el resultado al usuario.

💼 Aplicación Empresarial

Las organizaciones utilizan estimaciones para tomar decisiones sobre producción, inventarios y estrategias de venta. Este sistema permite obtener una proyección rápida basada en información histórica.

Ejemplo

Mes Ventas
1 1000
2 1500
3 2000 (estimado)

🎯 Objetivo del Proyecto

Desarrollar una aplicación web capaz de estimar ventas futuras mediante el método numérico de Interpolación Lineal, facilitando la toma de decisiones en entornos empresariales.
