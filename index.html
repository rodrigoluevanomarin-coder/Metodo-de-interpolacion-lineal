📈 Sistema de Predicción de Ventas

Aplicación basada en el método numérico de Interpolación Lineal

📚 Introducción

Las empresas necesitan predecir sus ventas futuras para planificar inventarios, compras y estrategias comerciales. Este proyecto utiliza el método numérico de Interpolación Lineal para estimar valores futuros a partir de datos históricos conocidos.

🧮 ¿Qué es la Interpolación Lineal?

La interpolación lineal es un método numérico que permite estimar valores desconocidos utilizando dos puntos conocidos. Se asume que el comportamiento entre ambos puntos sigue una línea recta.

y = y₁ + ((x - x₁)(y₂ - y₁)) / (x₂ - x₁)

💼 Aplicación Empresarial

Una empresa conoce sus ventas de los primeros meses del año y desea estimar las ventas de meses futuros para mejorar la toma de decisiones.

Ejemplo

Mes Ventas
1 1000
2 1500
3 (estimado) 2000

🎯 Objetivo del Proyecto

Desarrollar una aplicación web capaz de estimar ventas futuras utilizando Interpolación Lineal, permitiendo a las empresas realizar proyecciones rápidas y sencillas.
